### Teconologia, Engenharia de Software e Dados Aplicados 🚀

Olá! Sou **Engenheiro de Software** focado no desenvolvimento de sistemas corporativos de alta integridade e pipelines de dados complexos. Meu trabalho une o rigor da arquitetura de software backend com o poder analítico da ciência de dados.

Atuo construindo soluções escaláveis e transações seguras no ecossistema corporativo, utilizando meu background científico como uma vantagem estratégica para sistemas complexos.

---

## 💻 1. Engenharia de Software & Desenvolvimento Backend

Meu foco principal de atuação e desenvolvimento de soluções robustas:
* **Arquitetura Corporativa:** Desenvolvimento de APIs RESTful estruturadas, microsserviços e implementação do padrão BFF (Backend for Frontend).
* **Gestão de Transações:** Experiência prática na orquestração de fluxos complexos em ERPs (Apache OFBiz), garantindo transações atômicas imunes a falhas (*rollback-safe*).
* **Modelagem de Dados:** Criação de modelos relacionais desacoplados, focando em alta disponibilidade e consistência de dados (PostgreSQL/MySQL).

---

## 📊 2. Ciência de Dados & Quimioinformática

Desenvolvimento de pipelines analíticos e automação de fluxos de dados massivos:
* **Engenharia de Dados:** Manipulação, limpeza e transformação de dados analíticos complexos utilizando Python e Pandas.
* **Computação Científica:** Automação de triagens moleculares rápidas (*Molecular Screening*) e predições ADMET utilizando a biblioteca RDKit.
* **Validação Estatística:** Aplicação de modelos bioestatísticos avançados e inferência causal para integridade de dados públicos e privados.

---

## 🔬 3. Ciências da Saúde & Domínio de Negócio

Minha base científica atua como um acelerador de domínio para HealthTechs e sistemas de biotecnologia:
* **Mestre em Ciências da Saúde** (Universidade São Francisco) com foco em metabolômica e espectrometria de massas.
* **Graduado em Biomedicina**, unindo a compreensão de requisitos clínicos complexos à arquitetura de softwares médicos e laboratoriais.
